About The Project


Vibetune leverages AI to create highly personalized music playlists tailored to your current mood, activities, and preferences. With real-time mood analysis and Spotify API integration, Vibetune ensures that the music you hear is always in tune with your vibe.

Key Features:

AI-powered playlist generation based on real-time mood inputs.
Integration with Spotify for seamless playback.
Cross-platform support with a sleek, user-friendly interface.
